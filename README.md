# Haulvisor

Route optimization platform for owner-operator truck drivers. Analyze every route, calculate real profit, and find the fastest way home.

## Features

- Multi-stop route optimization with profit analysis
- Round-trip planning with "Home By" scheduling
- Real-time cost modeling (fuel, maintenance, daily costs)
- Lane watchlists and performance tracking

## Development

```bash
npm install
npm run dev
```
